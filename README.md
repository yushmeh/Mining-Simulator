# ⛏ Mining Simulator — Интерактивный симулятор майнинга

[![Flask](https://img.shields.io/badge/Flask-latest-000000?style=for-the-badge&logo=flask&logoColor=white)](https://flask.palletsprojects.com)

**Mining Simulator** — интерактивное игровое веб-приложение на **Flask**. Игрок добывает полезные ископаемые, прокачивает инструменты, расширяет рюкзак и зарабатывает внутриигровую валюту. Прогресс сохраняется между сессиями.

## 🎮 Игровой процесс

### ⛏ Добыча ресурсов
- **Реалистичный кликер** — визуальная отдача при ударе: эффект покачивания руды.
- **7 видов руды** с уникальными параметрами редкости, прочности (HP) и стоимости:

| Руда | Редкость | Особенность |
| :--- | :---: | :--- |
| 🪨 Камень | Обычная | Базовый ресурс |
| 🪨 Уголь | Обычная | — |
| ⚙️ Железо | Необычная | — |
| 💎 Сапфир | Редкая | Высокая стоимость |
| 💎 Рубин | Редкая | — |
| 💜 Аметист | Эпическая | — |
| 💎 Алмаз | Легендарная | Максимальная стоимость |

### 🎨 Динамический интерфейс
- Цветовая гамма, неоновое свечение и полоска HP автоматически меняются в зависимости от текущей добываемой руды.

### 🎒 Инвентарь и магазин
- **Рюкзак** — наглядное отображение собранных ресурсов с ограничением по слотам.
- **Магазин** — улучшения кирок и расширение рюкзака за заработанные монеты с мгновенным обновлением параметров.
- **Уведомления** — всплывающие сообщения о добыче, покупках и переполнении рюкзака прямо внутри игрового интерфейса.

## 🏗 Архитектура и классы

Проект построен на разделении бэкенд-логики (Python/Flask) и интерактивного отображения (HTML/CSS/JS). Каждый игровой объект инкапсулирован в отдельный класс:

| Класс | Описание |
| :--- | :--- |
| `Player` | Баланс монет, инвентарь, экипированная кирка, активный камень. Хранит состояние сессии. |
| `Stone` | Генерирует параметры руды: прочность (HP), базовую стоимость, название и системный ID. |
| `Pickaxe` | Модификатор инструмента: сила удара (урон по HP) и стоимость улучшения. |
| `Inventory` | Следит за лимитом слотов, считает предметы, добавляет ресурсы. |
| `Upgrade` | Логика прокачки уровней кирки и расширения вместимости рюкзака. |
| `Game` | Управляющий класс: шансы выпадения руды, транзакции магазина, калькуляция прибыли. |

## 📂 Структура проекта

```text
├── app/
│   ├── __init__.py        # Инициализация Flask-приложения
│   ├── routes.py          # Обработка маршрутов страниц
│   ├── models.py          # Модели данных (Player, Stone, Pickaxe, ...)
│   └── game_logic.py      # Ключевая логика игрового процесса
│
├── static/
│   ├── css/
│   │   └── style.css      # Стили, темы руд и анимации
│   ├── js/
│   │   └── main.js        # Асинхронное взаимодействие и обновление UI
│   └── images/            # Графические текстуры и картинки руд
│
├── templates/
│   └── index.html         # Главный шаблон игрового интерфейса
│
├── config.py              # Конфигурационные параметры Flask
├── requirements.txt       # Зависимости проекта
└── run.py                 # Точка входа для запуска сервера
```

## 🛠 Инструкция по запуску

Для работы необходим **Python 3.14+**.

```bash
# 1. Установить зависимости
pip install -r requirements.txt

# 2. Запустить сервер
python run.py

# 3. Открыть в браузере
# http://127.0.0.1:5000
```
