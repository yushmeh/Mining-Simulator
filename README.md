**Quantum Miner** — интерактивное игровое веб-приложение на **FastAPI + Pydantic v2**. Игрок добывает полезные ископаемые, прокачивает инструменты, расширяет рюкзак, следит за рынком и зарабатывает внутриигровую валюту. Поддерживается система престижа и мини-игры.

## 🎮 Игровой процесс

### ⛏ Добыча ресурсов
- **Кликер с физикой удара** — анимация тряски, визуальные критические удары и взрывы.
- **18 видов руды** от Камня до Алмаза с уникальными параметрами редкости, прочности и стоимости.
- **Уровневая система** — новые руды разблокируются по мере роста уровня игрока.

### 🎒 Инвентарь и магазин
- **Рюкзак** — ограниченная вместимость, расширяется через апгрейды.
- **8 апгрейдов** — от заточки кирки до магнита и авто-буровой.
- **Авто-бур** — добывает руды каждую секунду в фоне.

### 📈 Рынок
- Цены на все руды **обновляются каждые 5 минут**, колеблясь от ×0.5 до ×2.0 от базовой стоимости.
- Виджет рынка в левой панели показывает актуальные изменения в реальном времени.

### ⭐ Престиж
- При достижении 10 уровня и накоплении монет доступен **сброс прогресса**.
- За каждый престиж — постоянный бонус ×0.5 к удаче и сохранение 10% монет.

### 🎰 Мини-игры
При добыче руды с шансом 2.5% активируется одна из 6 мини-игр:

| Мини-игра | Описание |
| :--- | :--- |
| 🏃 Бегущая руда | Прыгай через динамиты — ПРОБЕЛ или клик |
| 🚪 Три тоннеля | Выбери из трёх проходов, где спрятан сундук |
| 🎯 Точный удар | Останови бегущий курсор в зелёной зоне |
| 💣 Сапёр | Открой 5 безопасных клеток из 16 |
| ⚡ Быстрые клики | Нажми кнопку 15 раз за 3 секунды |
| 🔢 Числовой сейф | Угадай 3-значный код за 4 попытки |

## 🆕 Нововведения

### Новые апгрейды

| Апгрейд | Уровень | Эффект |
| :--- | :---: | :--- |
| 💥 Взрывчатка | 5 | Каждый N-й удар наносит ×10 урона |
| 🔭 Сканер руд | 5 | Показывает следующую руду до добычи |
| ⚗️ Плавильня | 6 | Авто-продажа дешёвых руд после добычи |
| 🧲 Магнит | 7 | Шанс получить 2 руды за один удар |

### Архитектурные изменения
- **Flask → FastAPI** — нативная async поддержка, автодокументация `/docs` в dev-режиме.
- **Pydantic v2** — все модели (`Player`, `Stone`, `RockConfig`, `Achievements`) на `BaseModel`.
- **Clean Architecture** — игровая логика вынесена в чистый слой `game.py` без HTTP-зависимостей.
- **pydantic-settings** — конфигурация через `.env` файл.
- **Sci-Fi Dark Mode UI** — переработан интерфейс: шрифты Rajdhani + Share Tech Mono, CSS-переменные, CRT-эффект.

## 🏗 Архитектура

Проект построен на **3-уровневой архитектуре**:

| Слой | Файл | Описание |
| :--- | :--- | :--- |
| Конфигурация | `core/config.py` | Settings через pydantic-settings |
| Модели | `core/models.py` | Pydantic v2: Player, Stone, RockConfig, Achievements |
| Логика | `core/game.py` | Чистые stateless функции без HTTP-контекста |
| API | `core/routes.py` | Тонкий слой FastAPI: валидация → логика → ответ |
| Точка входа | `main.py` | FastAPI app factory + монтирование статики |

```text
│
├── templates/
│   └── index.html         # Jinja2 шаблон игрового интерфейса
│
├── main.py                # Точка входа (uvicorn)

```

## 🌐 API эндпоинты

| Метод | Путь | Описание |
| :---: | :--- | :--- |
| `GET`  | `/` | Главная страница |
| `POST` | `/api/click` | Удар киркой по руде |
| `POST` | `/api/auto_mine` | Тик авто-бура |
| `POST` | `/api/sell` | Продать весь инвентарь |
| `POST` | `/api/upgrade` | Купить апгрейд (`type: power\|luck\|bag\|auto\|explosive\|scanner\|smelter\|magnet`) |
| `POST` | `/api/prestige` | Выполнить престиж |
| `POST` | `/api/market/update` | Обновить рыночные цены |
| `POST` | `/api/minigame/result` | Засчитать результат мини-игры |

## 🛠 Инструкция по запуску

Для работы необходим **Python 3.12**.

```bash
# 1. Создать виртуальное окружение
py -3.12 -m venv .venv
.venv\Scripts\activate  # Windows
# source .venv/bin/activate  # Linux/Mac

# 2. Установить зависимости
pip install -r requirements.txt

# 3. Запустить сервер
uvicorn main:app --reload

# 4. Открыть в браузере
# http://127.0.0.1:8000
# Swagger UI (dev): http://127.0.0.1:8000/docs
```
